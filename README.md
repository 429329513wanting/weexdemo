# 使用方法
  基于weexpack脚手架构建，详细使用方法请参考 https://github.com/weexteam/weex-pack/tree/dev

### cd 到文件夹
### npm install
### weexpack run ios

# 知识点
    .使用自带的navigator模块跳转页面
    .与原生交互
    .自定义组件
    .页面间传值
    .处理weex不能直接加载本地图片局限
    .扩展weex加载网络图片协议

### ps 扫码只是适配了iOS端。需要三端扩展。可以自己用weexpack创建个新工程，把src文件夹替换过去。再按照weexpack命令打包和运行
